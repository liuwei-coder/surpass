Java工程师学习资料，帮助学习者掌握开发基本知识，承接web应用的后端设计开发。

一共四讲，适合两天的训练营。

## 学员要求

本培训不是零基础的，要求学员具备互联网开发的基本知识。

- HTML
- JavaScript
- CSS
- 命令行

以上技术，只要懂得基本语法和如何使用即可。

## 环境准备

参加培训之前，请按照[文档](docs/preparation.md)安装软件，做好环境准备。
